HTML Structure & Semantics

The Foundation of the Web

HTML (HyperText Markup Language) is not a programming language but a markup language. It tells the browser how to structure content. Every web page you visit is built upon an HTML skeleton.

Essential Definitions

<h1>
Headings

Headings range from h1 (Most Important) to h6 (Least Important). Search engines use these to understand your content.

<img>
Images

A void element (no closing tag). Uses the src attribute to find the file and alt for description.

Module 2

CSS Styling & Layouts

How CSS Works

CSS (Cascading Style Sheets) applies rules to HTML elements. The browser loads the HTML, then paints it according to your CSS rules.

The Box Model

Every element is a box. Understanding this is key to layout.

Margin
Border
Padding
Content

Figure 2.1: CSS Box Model

Selectors Deep Dive

  • p { } Element Selector

    Targets ALL paragraphs on the page.

  • .btn { } Class Selector

    Targets any element with class="btn". Reusable!

  • #header { } ID Selector

    Targets the unique element with id="header".
